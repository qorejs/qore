## 📋 执行摘要

Qore v1.0 是 Qore 框架的首个正式生产版本，标志着框架从 POC 阶段进入成熟可用阶段。本文明确定义 v1.0 的核心功能范围、发布标准和版本节奏，确保发布质量可控、预期清晰。

### 核心定位

> **AI-Native 前端框架** —— 为流式 AI 应用而生的轻量级响应式框架

### 版本目标

- ✅ 提供稳定、完整的信号响应式系统
- ✅ 支持 AI 流式渲染的一等公民支持
- ✅ 完整的 SSR 能力，支持边缘部署
- ✅ 开发者友好的 API 和工具链
- ✅ 生产级测试覆盖和文档

## ❌ v1.0 不包含的功能

以下功能明确**不在** v1.0 范围内，将在后续版本迭代：

| 功能 | 原因 | 计划版本 |
|------|------|----------|
| **移动端支持 (React Native)** | 需要额外架构设计 | v1.5+ |
| **完整插件系统** | 生态建设需要时间 | v1.2+ |
| **E2E 测试框架** | 优先级低于单元测试 | v1.1+ |
| **可视化编辑器** | 非核心功能 | v2.0+ |
| **企业级功能 (SSO, RBAC)** | 商业化功能 | v2.0+ |
| **多框架适配层** | 增加复杂度 | v1.5+ |
| **完整的国际化系统** | 社区贡献为主 | v1.2+ |
| **动画系统** | 非核心功能 | v1.3+ |
