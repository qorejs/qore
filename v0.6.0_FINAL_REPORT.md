## 🚀 使用示例

### Devtools

```typescript
import { createApp } from '@qore/core'
import { enableDevtools } from '@qore/devtools'

const app = createApp({ /* ... */ })

// 启用 devtools
const devtools = enableDevtools(app, {
  enableComponentTree: true,
  enableStateMonitoring: true,
  enablePerformance: true,
})

// 访问状态
const state = devtools.getState()
```

### 组件

```typescript
import { Button, Input, ToastProvider, success } from '@qore/primitives'

// ToastProvider 包裹应用
<ToastProvider position="top-right">
  <App />
</ToastProvider>

// 使用组件
<Button variant="primary" onClick={() => success('成功!')}>
  点击
</Button>

<Input
  label="邮箱"
  type="email"
  placeholder="请输入"
/>
```

### CLI

```bash
# 创建项目
pnpm create qore my-app

# 指定模板
pnpm create qore my-app --template full

# 开始开发
cd my-app
pnpm dev
```
